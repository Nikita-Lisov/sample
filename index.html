01. Getting Started

To customize this landing page template you just need the basic knowledge of html/css and a text editor.
Replace the default content with your own images and texts.

You can also include your own css and js code in 'static/custom.css' and 'static/custom.js'.

02. Navigation

Top menu

In top menu, 'href' attribute of each link should have this format: '#'+'id of the section you want to navigate to'.
Make sure that your section that you have included in navigation has 'id' attribute set up.

In top menu, you can also include external links.

Anchor links

If a 'div' element has 'id' attribute set up and you want to include a button or a link to navigate to it, just add class="scroll-link" and href="#your_element_id" to a button or a link.

05. Reviews

Html files that are in the template package have the 'Reviews' section. All you need is simply replace the default quotes with your content and replace authors' names and pictures. It all can be done in .html file and requires beginner's knowledge of html/css.

To make transitions we use Slick JS.
Documentation of slick.js: http://kenwheeler.github.io/slick/

06. Modal windows

You can include as many modal windows on a page as you want.
Create a 'div' element with the style attribute display="0" and additional data-modal-content="1" (1 is id of your modal window). Put custom content in this 'div' element.
Now, you have modal window created, just add attribute data-modal="1" to any button or link in your html (where "1" is id of modal window).

08. Animations

There are some prebuilt animations to load the content. Just assign one of these classes to each element you want to have an on-load animation:

  • 'animate-show'
  • 'animate-from-up'
  • 'animate-from-left'
